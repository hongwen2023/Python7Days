# 假设台账（Assumptions Ledger）

> 记录所有"因为模型/工具做不到 X，所以我才做 Y"的假设。
> 每月 re-baseline 仪式逐条重新验证；每次 Claude Code 大版本或新模型发布后，优先复验相关条目。
> **被推翻的假设不删行**，状态改为"已失效"并写明失效原因——它们是进步的刻度。

状态取值：`成立` / `存疑` / `已失效`

## 使用说明

- 新增假设：随手加一行，日期写当月即可。
- 复验：改"最近验证"和"状态"；失效的在备注写明是哪个版本/模型让它失效。
- 每条"已失效"都对应一次删除动作（删配置、删 hook、删习惯），删完在备注打钩。
